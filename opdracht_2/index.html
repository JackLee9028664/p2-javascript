Wat is het verschil met de script tag boven in te laden in de head tag t.o.v onder inladen aan het einde van je HTML tag?

verschil is dat in de head de script als eerst laad, het lijkt dan dat de script traag is, en dat in de body de pagina eerst laat, en dan laat de script

Wat doet de attribuut “defer” in de script tag?

zorgt dat de javascript pas na de hmtl document wordt geparsed, maar well alvast gedownload wordt

Wat is het verschil tussen HTML en JavaScript?

JavaScript is een programmeertaal die wordt gebruikt om dynamische en interactieve webpagina's te maken; HTML is een opmaaktaal die wordt gebruikt om inhoud te structureren en statische webpagina's te maken.
